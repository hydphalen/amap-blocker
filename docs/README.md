## 文档规范

### 格式规范

1. 使用 Markdown 格式
2. 中文编写
3. 代码示例使用围栏代码块
4. 表格使用 Markdown 表格语法

### 内容规范

1. 文档内容准确、完整
2. 代码示例可运行
3. 命令示例可复制
4. 链接有效

### 更新规范

1. 功能变更时更新文档
2. 定期检查文档准确性
3. 及时修复错误
