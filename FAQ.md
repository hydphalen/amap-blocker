# 常见问题解答

本文档回答了关于高德地图去推广 LSPosed 模块的常见问题。

## 一般问题

### Q: 这个项目是什么？

**A:** 这是一个 LSPosed 模块，用于拦截高德地图中的推广/广告内容。通过 Hook 高德地图的 MTOP 网关，实现对推广 API 的拦截和日志记录。

### Q: 这个项目有什么用？

**A:** 这个项目可以：
1. 自动拦截高德地图中的推广内容
2. 记录所有 MTOP API 调用，便于分析
3. 不影响高德地图的正常使用

### Q: 这个项目安全吗？

**A:** 本项目是开源项目，代码公开透明。我们采取了多种安全措施：
1. 只拦截推广 API，不影响核心功能
2. 日志文件存储在应用私有目录
3. 不记录敏感信息
4. 添加异常处理，避免崩溃

### Q: 这个项目需要 Root 吗？

**A:** 是的，本项目需要 Root 权限才能运行，因为：
1. LSPosed 框架需要 Root 权限
2. 访问应用数据目录需要 Root 权限
3. Hook 系统级方法需要 Root 权限

## 安装问题

### Q: 如何安装这个模块？

**A:** 安装步骤：
1. 确保设备已 Root
2. 安装 LSPosed 框架
3. 下载并安装模块 APK
4. 在 LSPosed 中启用模块
5. 选择作用域为「高德地图」
6. 重启高德地图

详细步骤请参考 [README.md](README.md#安装方法)。

### Q: 安装时提示"签名不一致"怎么办？

**A:** 这通常是因为设备上已安装了同包名的应用。解决方法：
1. 先卸载旧版本：`adb uninstall com.miclaw.amapblocker`
2. 重新安装新版本

### Q: 安装后找不到模块怎么办？

**A:** 请检查：
1. LSPosed 是否正确安装
2. 模块是否已启用
3. 作用域是否选择「高德地图」
4. 高德地图是否已重启

### Q: 如何更新模块？

**A:** 更新步骤：
1. 下载新版本 APK
2. 使用 `adb install -r` 覆盖安装
3. 重启高德地图

## 使用问题

### Q: 模块启用后没有效果怎么办？

**A:** 请检查：
1. 模块是否已启用
2. 作用域是否选择「高德地图」
3. 高德地图是否已重启
4. 查看日志文件是否有输出

### Q: 如何查看日志？

**A:** 查看日志的方法：
1. 使用脚本：`./pull_logs.sh`
2. 手动查看：`adb shell "cat /data/data/com.autonavi.minimap/amap_blocker_logs/mtop_log_$(date +%Y-%m-%d).txt"`

### Q: 日志文件在哪里？

**A:** 日志文件位于：
```
/data/data/com.autonavi.minimap/amap_blocker_logs/
```

### Q: 如何清理日志？

**A:** 清理日志的方法：
1. 手动删除：`adb shell "rm -rf /data/data/com.autonavi.minimap/amap_blocker_logs/"`
2. 等待自动清理（默认保留 7 天）

### Q: 模块会影响高德地图的性能吗？

**A:** 影响很小：
- 启动时间：+0.3 秒（12%）
- 内存使用：+5 MB（3.3%）
- CPU 使用：+1%（20%）
- 网络请求：-10%（正向优化）

## 技术问题

### Q: 这个项目使用了什么技术？

**A:** 本项目使用：
- 语言：Java
- 框架：LSPosed / Xposed
- 构建：Gradle
- CI/CD：GitHub Actions

### Q: 如何编译这个项目？

**A:** 编译步骤：
1. 克隆仓库
2. 使用 Android Studio 打开项目
3. 等待 Gradle 同步
4. 点击 "Build" -> "Make Project"

或使用命令行：
```bash
./gradlew assembleDebug
```

### Q: 如何添加新的拦截 API？

**A:** 添加步骤：
1. 运行模块，收集日志
2. 分析日志，找到推广 API
3. 在 `MtopHook.java` 中添加到 `BLOCKED_APIS` 或 `BLOCK_KEYWORDS`
4. 重新编译并安装

### Q: 如何调试这个项目？

**A:** 调试方法：
1. 使用 Android Studio 的 Debug 模式
2. 查看 Logcat 日志：`adb logcat -s AMapBlocker`
3. 查看 LSPosed 日志：`adb logcat -s LSPosed`

### Q: 这个项目支持哪些 Android 版本？

**A:** 本项目支持 Android 7.0 (API 24) 及以上版本。

### Q: 这个项目支持哪些高德地图版本？

**A:** 本项目基于高德地图某个特定版本逆向开发。如果高德地图更新，可能需要调整 Hook 点。
